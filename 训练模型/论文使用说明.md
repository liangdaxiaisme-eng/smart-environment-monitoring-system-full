# 论文数据和系统说明

## 一、论文中使用的数据

### 1. 大气监测数据
- **来源**: UCI 机器学习库 - 北京多站点空气质量数据
- **链接**: https://archive.ics.uci.edu/dataset/501/beijing+multi+site+air+quality+data
- **时间范围**: 2013年3月 - 2017年2月
- **数据量**: 42万+条记录，12个监测站点
- **特征**: PM2.5, PM10, SO2, NO2, CO, O3, 温度, 气压, 露点, 降水, 风速 (11维)
- **用途**: LSTM时序预测、GNN图神经网络训练

### 2. 水质分类数据
- **来源**: 基于大气数据生成的水质特征图
- **样本量**: 2000张
- **类别**: 4类 (清洁/轻度污染/中度污染/重度污染)
- **生成方式**: 将11维特征转换为11×11灰度图像，再扩展为3通道
- **用途**: ResNet18+SE图像分类模型训练

### 3. 多模态融合数据
- **来源**: 大气监测数据 + 模拟图像特征
- **处理**: 时序特征 + 图像特征配对
- **用途**: 交叉注意力融合模型训练

## 四、模型使用说明

# 加载模型
model = torch.load('lstm_best.pth', map_location='cpu')
model.eval()

# 输入: 24小时历史数据 (batch_size, 24, 11)
# 输出: 未来PM2.5预测值
input_data = torch.randn(1, 24, 11)

    prediction = model(input_data)
